# BP Tracker EDA

Projeto pessoal para análise exploratória de dados de medições de pressão arterial coletadas ao longo de duas semanas.

Objetivos:

- praticar Python e análise exploratória de dados
- analisar variações entre medições de manhã e noite
- gerar visualizações simples
- extrair insights sobre tendência e variabilidade

Stack utilizada:

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
